# MDM Python Parser

Helper scripts for processing HWP and PDF files.

## Setup

```bash
python -m venv venv
source venv/bin/activate  # or venv\Scripts\activate on Windows
pip install -r requirements.txt
```

## Usage

### HWP to SVG

```bash
python hwp_to_svg.py input.hwp output.svg
```

### PDF Processor

```bash
python pdf_processor.py input.pdf
```
